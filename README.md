# Telephone Dictionary System

A professional web application built with Laravel to manage contacts, groups, and more.

## Features

- User Authentication (Register, Login, Logout)
- Contact Management (CRUD)
- Group Management (CRUD)
- Dashboard with statistics
- AJAX-powered live search
- Mark/unmark contacts as favorite
- CSV Import/Export for contacts
- REST API for contact management

## Tech Stack

- Laravel
- MySQL
- Blade Templates
- Bootstrap 5
